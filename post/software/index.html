记录个人常用的软件

Application

Typora: Typora is a minimal markdown editor, providing new ways for reading and writing markdown.

iTerm2: iTerm2 is a replacement for Terminal and the successor to iTerm.

Snipaste: Snipaste is a simple but powerful snipping tool, and also allows you to pin the screenshot back onto the screen.

Notion: All-in-one workspace.

Dash: Dash gives your Mac instant offline access to 200+ API documentation sets.

Alfred: Alfred is an award-winning app for macOS which boosts your efficiency with hotkeys, keywords, text expansion and more.

Magnet: Window manager for Mac.

Karabiner-Elements: A powerful and stable keyboard customizer for macOS.

Maccy: Lightweight clipboard manager for macOS.

Go2Shell: Opens a terminal window to the current directory in Finder.

AltTab: AltTab brings the power of Windows’s “alt-tab” window switcher to macOS.

WebTorrent: WebTorrent is a streaming torrent client for the web browser and the desktop.

Barrier: Open-source KVM software.

Vivaldi: he new Vivaldi browser protects you from trackers, blocks unwanted ads, and puts you in control with unique built-in features.

FireFox: Get the browser that protects what’s important.

Google Chrome: Now more simple, secure, and faster than ever - with Google’s smarts built-in.

QSpace: QSpace is a clean and efficient Multi-view File Manager, also can be a FTP/SFTP/WebDAV client.

foobar2000: foobar2000 is an advanced freeware audio player for the Windows platform.

IINA: The modern media player for macOS.

Motrix: A full-featured download manager.

Postman: The Collaboration Platform for API Development.

Visual Studio Code: Code editing. Redefined.

eZip: 为 macOS 而设计的苹果系统压缩软件！

Charles: Charles is an HTTP proxy / HTTP monitor / Reverse Proxy that enables a developer to view all of the HTTP and SSL / HTTPS traffic between their machine and the Internet.

qBittorrnet: The qBittorrent project aims to provide an open-source software alternative to µTorrent.

AnyDesk: Remote Desktop Software That Actually Works.

LICEcap: simple animated screen captures.

Hidden Bar: An ultra-light MacOS utility that helps hide menu bar icons.

Stats: macOS system monitor in your menu bar.

Command Line Tool

Non-Built-In

bat: A cat(1) clone with wings.

trash-cli: Command line interface to the freedesktop.org trashcan.

httpie: Modern command line. HTTP client – user-friendly curl alternative with intuitive UI, JSON support, syntax highlighting, wget-like downloads, extensions, etc.

jq: Command-line JSON processor.

fx: Command-line JSON processing tool.

fzf: A command-line fuzzy finder.

tig: Text-mode interface for git.

mycli: A Terminal Client for MySQL with AutoCompletion and Syntax Highlighting.

lrzsz: lrzsz is a unix communication package providing the XMODEM, YMODEM, ZMODEM file transfer protocols.

ncdu: Ncdu is a disk usage analyzer with an ncurses interface.

tmux: tmux is a terminal multiplexer.

ffmpeg: A complete, cross-platform solution to record, convert and stream audio and video.

rg: ripgrep recursively searches directories for a regex pattern.

tree: Tree is a recursive directory listing command that produces a depth indented listing of files, which is colorized ala dircolors if the LS_COLORS environment variable is set and output is to tty.

shellcheck: ShellCheck, a static analysis tool for shell scripts.

htop: An interactive process viewer for Unix.

glances: Glances is a cross-platform system monitoring tool written in Python.

pm2: Advanced, production process manager for Node.js.

pandoc: If you need to convert files from one markup format into another, pandoc is your swiss-army knife.

navi: An interactive cheatsheet tool for the command-line.

graphviz: Graphviz is open source graph visualization software.
